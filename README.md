## 1.Config fb, google login, email for development environment
1. **Create file *`.env.development`* at the root of project**  
2. **Set value for these followings var**
```
FB_APP_ID = 
FB_APP_KEY = 

GOOGLE_OAUTH2_ID = 
GOOGLE_OAUTH2_KEY =

MAIL_ADDRESS = 
MAIL_SMTP_PORT = 
MAIL_DOMAIN = 
MAIL_USERNAME = 
MAIL_PASSWORD = 
```
